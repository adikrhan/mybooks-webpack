# mybooks-webpack
MyBooks web app created using webpack.

Run 'npm install' in the master folder and then 'npm run start' to automatically open the app in the browser. 

Functionality implemented this far:
- Search for books with the help of Goodreads API.
- Add and delete books from Read- or Wishlist and Bookshelf.
- Rate books added to Read-list.
- Write, update and delete reviews and blogposts.
- Update feed on the home page with the latest reviews and blogposts. 

The app uses Local Storage to store booklists and blogposts. Sign up and sign in not yet implemented. 

Not checked for Cross-browser compatibility yet, developed using Chrome. 

Somewhat responsive to different screen sizes, but not 100% yet.
